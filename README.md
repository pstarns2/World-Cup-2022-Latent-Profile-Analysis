# World-Cup-2022-Latent-Profile-Analysis
Model-based clustering (tidyLPA/mclust) of 576 World Cup players on 6 event based metrics. Why LPA: cluster count and covariance structure are selected by fit indices (BIC, bootstrapped LRT, entropy) instead of guessed; assignments come with posterior probabilities. The model recovered 5 interpretable roles without ever seeing positions.
